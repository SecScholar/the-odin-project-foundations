This html file is an example of one boilerplate html file that I modified. To be specific, in this example, was a base shell / template homepage for the webpage. It's become the main webpage because whenever you create a website, the main page runs on index.html. In the index.html, I modified and left some practical coding examples of different types of elements. Elements will take you to different subdomains and sections of the internet. Think of a boilerplate code like a template for web application coding, it won't have any text or elements, it will only have tags for elements, that you will eventually fill in with your intended coded application. 
I'll put prime examples of each concept in the md for notes sake.
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Another webpage</title>
</head>
<body>
</body>
</html>
